# QUICKSTART

## Get it running in 2 steps:

### Linux/Mac:
```bash
chmod +x setup.sh run.sh
./setup.sh
./run.sh
```

### Windows:
```cmd
setup.bat
run.bat
```

### Manual install:
```bash
python -m venv venv

# Windows:
venv\Scripts\activate

# Linux/Mac:
source venv/bin/activate

pip install -r requirements.txt
python main.py
```

---

## What you'll see:

A window filled with colored cells that live, die, evolve, and compete.

Each color is a different species.

Just watch the patterns emerge.

---

## Controls:

- **SPACE** = pause
- **1-5** = speed (1x to 100x)
- **S** = show/hide stats
- **R** = reset world
- **Q** = quit
